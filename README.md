## Deskripsi Singkat Proyek

**Berita Cihuy** adalah website portal berita sederhana yang memungkinkan pengguna untuk:
- Melihat daftar berita dengan tampilan seperti portal berita modern.  
- Membaca detail berita lengkap beserta komentar pengunjung.  
- Menambahkan berita baru lengkap dengan kategori dan gambar.  
- Mengelompokkan berita berdasarkan kategori seperti *Teknologi, Olahraga, Hiburan,* dll.  
- Memberi komentar di setiap berita.

Website ini menampilkan data dari database dengan relasi antara:
- **Wartawan**  
- **Berita**  
- **Komentar**  
- **Kategori**
